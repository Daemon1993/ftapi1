Fcoin

文档来源:https://developer.fcoin.com/zh.html#45fa4e00db

#####配置

```
打开config.py
填写fcoin的api.key
填写fcoin的api.secret
```

##### 运行

```
这个脚本需要python2或者python3环境
然后直接运行python robot.py就可以了
```
